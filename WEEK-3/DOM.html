What is DOM and its working ?

  • The DOM is an application programming interface(API) for manipulating HTML and XML documents.
  • This manipulation can be adding, removing, and modifying parts of the HTML document.
  • Understanding DOM concept will give us an insight of the use of JS in making changes in our website.
  • When a webpage is loaded, the browser creates a tree-like structure called as DOM of the page.
  • The tags in HTML became the objects in DOM.
  • So, those objects can be manipulated by using inbuilt methods and properties.
